# Target Separator

Aim Training analysis requires information that aim trainers simply do not provide, coming in the formats of comma separated value (CSV files). Simple log files from applications such as Kovaaks or AimLabs prevent further analysis from machine learning models, necessitating the usage of video analysis to provide the most useful insight. This set of files aims to achieve finding a way to make the video useful to be feeded into a analysis model.

Such problems arise in this task:
1. Differentiating target
2. Differentiating crosshair
3. Removing frames that indicate end of task (menus and other screens)
